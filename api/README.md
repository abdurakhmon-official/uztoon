1. npx prisma init - to init prisma
2. npx prisma format - to format prisma/schema.prisma file
3. npx prisma migrate dev --name init - to migrate DB
4. npx prisma migrate reset - reset db
5. npx prisma studio - see db using prisma
6. npx ts-jest config:init - create a config file for jest
7. npm test - to test using jest
8. npx prisma db seed - to seed the db
9. npx prisma generate - generate prisma client
